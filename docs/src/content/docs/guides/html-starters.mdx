A single HTML file that loads matchina and a view library from a CDN. No bundler, so no JSX (React needs [its own page](/matchina/guides/react)). For adapter patterns, see [Frontend Integration Overview](/matchina/guides/integration-patterns).

## Imports

Demos load from **esm.sh**. For other CDNs, see [Use from a CDN](/matchina/guides/installation#use-from-a-cdn).

## Progressive Enhancement

:::tip[Try it with JS off]
Disable JS and open each demo. The **Progressive** column says what you'll see.
:::

## Considerations

- **Static HTML.** Need the card to render before JS runs? Stimulus, Vanilla, Alpine, petite-vue, Vue 3.
- **Footprint.** ~1.5 KB (VanJS) to ~40 KB (Vue 3).
