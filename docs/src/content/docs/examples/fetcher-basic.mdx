## About This Example

This example demonstrates a basic promise machine for handling asynchronous operations:

- Defines a state machine with Idle, Pending, Resolved, and Rejected states
- Manages promise lifecycle automatically
- Provides type-safe transitions between states
- Demonstrates how to handle errors and data results

The promise machine pattern is ideal for managing any asynchronous operation, from API calls to complex workflows with retry logic.
