## Theme Overview

Matchina uses a Catppuccin-inspired color system with carefully chosen palettes for both dark and light themes. The system emphasizes:

- **Accessibility** - WCAG AA compliant contrast ratios
- **Visual Hierarchy** - Clear semantic color mapping
- **Eye Comfort** - Reduced eye strain with soothing pastels
- **Consistency** - Unified across all visualizers

## Color Categories

### Semantic Colors
- **Backgrounds** - Base, mantle, crust for depth
- **Text** - Primary, secondary, muted for hierarchy
- **Surfaces** - Interactive elements and borders

### State Machine Colors
- **Active States** - Blue highlighting for current states
- **Ancestor States** - Transparent blue for hierarchy
- **Initial States** - Green for starting points
- **Final States** - Red for termination points

### Interactive Elements
- **Transitions** - Available, hover, recent, unavailable
- **Status** - Success, warning, error, info
- **Accents** - Purple, pink, teal for variety

## Implementation

The color system uses CSS custom properties for easy theme switching:

```css
:root[data-theme="mocha"] {
  --matchina-base: #1e1e2e;
  --matchina-text: #cdd6f4;
  --matchina-blue: #89b4fa;
  /* ... */
}
```

Components reference semantic variables:

```css
.state-active {
  background: var(--matchina-state-active);
  color: var(--matchina-bg-primary);
}
```

## Accessibility

All color combinations meet WCAG AA requirements:
- Normal text: 4.5:1 contrast minimum
- Large text: 3:1 contrast minimum  
- UI components: 3:1 contrast minimum

Colorblind-safe patterns and labels are used in addition to color coding.
