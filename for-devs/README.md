# For Developers - Quick Start

## Project Overview
Matchina is a TypeScript-first library for building type-safe state machines with powerful pattern matching. It's a lightweight, modular toolkit (3.42 kB full library gzipped) published to npm.

**Core Philosophy:**
- TypeScript-first with powerful type inference
- Nano-sized, opt-in primitives
- Composable APIs that work together or standalone
- Every state/transition is type-inferred with exhaustive pattern matching

Development Flow: Core → Tests → Examples → Docs
